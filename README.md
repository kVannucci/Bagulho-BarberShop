# 🪒 Bagulho BarberShop - Site Profissional para Barbearia

## 📋 Visão Geral
Bem-vindo ao **Bagulho BarberShop**, o site moderno e responsivo para a sua barbearia! Desenvolvido 100% com **HTML5, CSS3 e JavaScript puro** (sem frameworks ou backends), este projeto oferece uma experiência premium para clientes que buscam cortes impecáveis, barba perfeita e estilo único.

Ideal para promover serviços, agendamentos online (simulados via JS), galeria de trabalhos e contato rápido. Abra o `index.html` no navegador e veja a magia acontecer!

## ✨ Principais Funcionalidades
- **Design Responsivo**: Perfeito para mobile, tablet e desktop.
- **Cards de Serviços**: Destaque cortes, barba, coloração e mais com animações suaves.
- **Galeria Interativa**: Slider de fotos dos melhores trabalhos (CSS/JS).
- **Formulário de Agendamento**: Validação em tempo real e simulação de envio.
- **Efeitos Visuais**: Transições, hover effects e scroll suave.
- **Seções Completas**: Home, Sobre, Serviços, Galeria, Contato.

## 🛠️ Tech Stack
```
HTML5    CSS3     Vanilla JS
│        │         │
└── Sem dependências externas!
```

- **Zero Setup**: Apenas abra no browser.
- **Customizável**: Edite `style.css` para cores/logo, `script.js` para lógica extra.

## 🚀 Como Executar
1. Clone o repositório usando Git
2. Abra `index.html` no seu navegador favorito (Chrome, Firefox, etc.).
3. Ou use Live Server no VSCode para auto-reload.
